# ApolloClaim

ApolloClaim is a mobile application designed for claim management, providing features like claim submission, tracking, and management for users. It includes functionalities such as photo uploads, multi-step forms, filtering options, and more.

## Table of Contents
- [Features](#features)
- [Installation](#installation)

## Features
- Multi-step form for claim submission
- Upload and preview photos directly in the app
- Filter claims by status and date
- Track claims with real-time updates
- Export claim data in CSV/Excel formats

## Installation

   ```bash
   git clone https://github.com/username/apollo-claim.git
2. cd apollo-claim

4. expo start
